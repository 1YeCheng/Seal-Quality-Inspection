# Image Assets

Place public README images in this folder.

Recommended files:

```text
ui_screenshot.png          # Main desktop UI screenshot
2d_detection_example.jpg   # Visible-light annotated detection example
ir_detection_example.jpg   # Infrared annotated detection example
ng_trace_example.png       # NG record browser and traceability screenshot
log_window.png             # Log/CSV review screenshot
```

Before publishing, crop or blur QR codes, company names, brand names, file-system paths, and any private production information.
